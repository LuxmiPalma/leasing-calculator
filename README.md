# Leasing Calculator

A business-oriented web application built with Angular and TypeScript for calculating monthly leasing cost, total cost, interest impact and payment schedule.

## Features

- Calculate monthly leasing payments
- Include down payment and residual value
- Display total payable amount and total interest
- Generate month-by-month payment schedule
- Show remaining balance progress
- Responsive dashboard-style user interface

## Tech Stack

- Angular
- TypeScript
- HTML
- CSS

## Purpose

This project was created to strengthen my practical experience in modern frontend development, business logic, data presentation and user interface design.
